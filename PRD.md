# SqueezePic 产品需求文档 (PRD)

## 1. 产品定位
一个轻量、纯客户端、隐私安全的在线图片压缩与编辑工具，帮助用户快速减小图片体积，支持批量处理、格式转换和基础编辑。

## 2. 目标用户
- 前端开发者与设计师
- 网站内容编辑
- 社交媒体运营
- 任何需要临时压缩图片的普通用户

## 3. 核心功能 (V1.0)

### 3.1 图片上传
- 方式：点击上传按钮选择文件、拖拽图片至上传区域、从剪贴板粘贴。
- 格式支持：PNG, JPG, WebP, SVG（不压缩仅格式转换或尺寸调整）。
- 批量：一次可选多张，或在已有列表上追加。
- 限制：单文件最大 20MB，一次性最多 30 张，超出给出友好提示。

### 3.2 图片列表与管理
- 展示所有已添加图片的缩略图、原始文件名、原始大小、格式。
- 每张图片有独立的压缩设置（点击切换选中）或应用全局设置。
- 可删除单张、清空全部。

### 3.3 压缩与格式转换
- 质量滑块：范围 1-100，默认 80，实时预览压缩后大小（防抖计算）。
- 输出格式：保留原格式、或转换为 JPG、PNG、WebP。
- 批量应用：可勾选"应用到所有图片"，统一设置质量和格式。
- 预览对比：点击某张图片进入对比视图，左右上下对比原图与压缩结果，可缩放查看细节。
- **专业编码器（v1.1+）**：JPEG → MozJPEG、PNG → OxiPNG、WebP → libwebp，全部基于 WASM（@jsquash/*）。
  - 优势：在相同视觉质量下，体积稳定小于 Canvas `toBlob` 默认编码器；多次调整 quality/格式结果可预测，无"变大" bug。
  - 加载策略：首次使用前动态 import，初始化期间界面显示"正在加载编码器…"提示。

### 3.4 基础编辑（P2）
- 旋转：顺时针逆时针 90° 旋转，或自定义角度。
- 裁剪：自由比例裁剪、或固定比例（11, 43, 169），支持拖拽选区。
- 重置：撤销所有编辑，恢复原始图片。

### 3.5 下载
- 单张下载：每张图片旁有下载按钮，生成新文件名（如 `原文件名_compressed.jpg`）。
- 批量下载：全选或选中多张后，点击“打包下载”，生成 ZIP 文件，保留原文件结构（可选）。

## 4. 非功能性需求
- 隐私：所有图片仅在浏览器内存本地处理，永不离开用户设备。
- 性能：大图（5MB）压缩计算不能阻塞 UI，编码步骤使用 WASM 异步执行。
- 兼容性：主流现代浏览器（Chrome, Edge, Firefox, Safari 最新两个版本，需支持 `WebAssembly` 和 `ImageData`）。
- 响应式：桌面端为主，移动端自适应布局（垂直堆叠）。
- UIUX：暗色主题，简洁专业，操作引导清晰。
- **编码体积稳定性（v1.1+）**：所有格式（JPEG/PNG/WebP/保持原格式）经过专业编码器重新压缩后，体积稳定 ≤ 原图体积。 PNG 走 OxiPNG 无损（持平或缩小），JPEG/WebP 走有损质量参数。 SVG / GIF 等不支持格式自动降级到 Canvas `toBlob`。
- **临时保留 / 自动清理（v1.2+）**：所有图片仅在浏览器内存中保留，最多 **30 分钟自动清除**，关闭页面或标签时立即清除，无任何上传或持久化存储（无 localStorage / IndexedDB / Cache Storage / 后端）。
